# AI Ad Designer

A Claude Code plugin for designing, rendering, and launching paid social ads from a single chat window. HTML/CSS templates rendered to PNG via Playwright, full-ad image generation via Google Gemini (Nano Banana), and direct upload to Facebook / Instagram via the Meta Ads MCP connector.

Built for the **AI Ads Workshop**. No GitHub account required.

---

## What you can do here

- **Talk to Claude like a creative director.** Describe the campaign — ICP, offer, angle — and Claude scaffolds the campaign file, drafts the copy, picks a design lane, and renders the ad.
- **Four design lanes**, all driven by the same copy file:
  - **bold-text-designer** — typography-driven ads on solid color. ~40 templates.
  - **system-visual-designer** — the diagram IS the ad. Mechanism-as-message.
  - **chat-style-designer** — fake iMessage / Slack / Discord screenshots.
  - **gemini-image-designer** — full-ad photographic or illustrated images, generated by Google Gemini Nano Banana.
- **Push to Meta Ads in one sentence.** "Upload this ad to my Facebook ad account" → Claude uses the Meta Ads MCP to create the creative in your account, paused, ready for you to review in Ads Manager.
- **Add your own design lanes.** When you have a new ad format in mind, `/new-design-skill` scaffolds a brand-new skill following the workspace's structure.

## First-time setup

> If you just installed this plugin, run `/setup` in Claude Code and skip the rest of this section. The setup skill walks you through everything below interactively and verifies each step worked.

If you'd rather do it by hand, here's the checklist:

### 1. Install Node and Playwright

```bash
# From the workspace root
cd skills/shared
npm install
npx playwright install chromium
```

Requires Node.js 18+. Get it from https://nodejs.org if you don't have it.

### 2. Set your Gemini API key

Get a free key at https://aistudio.google.com/apikey.

```bash
# From the workspace root
cp .env.example .env
# Then open .env and paste your key into the GEMINI_API_KEY= line
```

Verify it works:

```bash
node skills/shared/tools/gemini-generate.js --check
```

You should see `✓ Gemini API key works`.

### 3. Connect the Meta Facebook Ads MCP

This is a Claude Desktop UI step — you do it in Settings, not in code:

1. Open Claude Desktop → click your name (bottom-left) → **Settings**.
2. Click **Connectors** in the sidebar.
3. Click the **+** button → **Add custom connector**.
4. **Name:** `Facebook Ads`
5. **URL:** `https://mcp.facebook.com/ads`
6. Click **Add** → sign in with the Meta account that owns your ad accounts.
7. Select the **Business Portfolio(s)** you want Claude to manage → **Authorize**.

To verify, ask Claude "list my Facebook ad accounts." It should return a list.

## Day-one walkthrough

Once setup is done, this is the fastest path from blank workspace to launched ad:

```
/new-campaign            → Claude asks ICP / offer / messaging questions,
                           creates campaigns/{slug}/CAMPAIGN.md

/new-ad                  → Claude asks which campaign + which design lane,
                           drafts COPY.md, renders the image

/upload-to-meta          → Claude uploads the finished PNG to your ad
                           account via the Meta Ads MCP (paused, for review)
```

Or just talk to Claude in plain English — "create a new campaign for my Pilates studio" or "build me a Gemini ad for the personal-injury campaign with a magazine-spread feel." The slash commands are shortcuts, not requirements.

## The cardinal rule of this workspace

**Copy first. Image second. Always.**

Every ad starts with `COPY.md` — the verbatim words the ad will say (headline, subtext, CTA). The image is just a render of the copy. Never generate an image and write copy to match it. This rule holds for the HTML lanes and for Gemini-generated ads. The design skills are interpretive — they read the copy and translate it into a visual.

This is the difference between *advertising* and *graphic design*. We're doing the former.

## Troubleshooting

| Symptom | Fix |
|---|---|
| `command not found: node` | Install Node.js 18+ from https://nodejs.org |
| `playwright install` fails | Network — retry, or install via `npx playwright install chromium --force` |
| `GEMINI_API_KEY is not set` | Check `.env` exists at workspace root and contains your key. Use `node skills/shared/tools/gemini-generate.js --check` to verify. |
| Gemini returns no image | Free tier sometimes rate-limits image generation. Wait 60s and retry. If it persists, enable billing at https://aistudio.google.com/billing. |
| Meta MCP "no ad accounts" | Re-open Claude Settings → Connectors → Facebook Ads → click Authorize and re-select Business Portfolios. Most "no accounts" errors are skipped Business Portfolio selection. |
| HTML render is blank | Open the `.html` file in a browser. If it renders there, re-run `npx playwright install chromium`. |
