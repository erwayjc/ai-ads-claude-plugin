You are creating a new ad in this workspace.

1. **Confirm the campaign.** If the user hasn't named one, ask which campaign (slug from `campaigns/`). If no campaigns exist, route them to `/new-campaign` first.
2. **Pick the design lane.** Offer the four options with one-line descriptions:
   - **bold-text-designer** — bold typography on solid color. The classic performance ad. ~40 templates to pick from.
   - **system-visual-designer** — the diagram IS the ad. Use when you want to *show* a mechanism.
   - **chat-style-designer** — fake chat screenshot (iMessage, Slack, Discord). Social proof via eavesdropping.
   - **gemini-image-designer** — full-ad photographic or illustrated image generated by Gemini. Use when the visual is doing emotional work.
3. **Draft `COPY.md` first** — headline, subtext, CTA. Copy comes before the image, always. If `COPY.md` already exists for this ad, use it verbatim.
4. **Invoke the chosen design skill** to render the image.
5. **QC** against the skill's checklist before reporting done.
